# Casino 67 Aura + Ego

Projeto em Python com Pygame feito por André B, Christian S e Rony F.

## Jogos incluídos

- Blackjack com dealer, hit, stand, double down e pagamento 3:2.
- Dice com apostas em baixo, sete exato, alto, par e ímpar.
- Roleta europeia 0-36 com número, cor, paridade, faixa e dúzias.
- Slots com 5 rolos, 3 linhas, 5 paylines e tabela de pagamentos.
- Mines com grade 5x5, quantidade de minas ajustável, multiplicador progressivo e saque.

## Como rodar

```bash
pip install -r requirements.txt
python main.py
```

Use o mouse para jogar. A tecla `ESC` volta ao menu; no menu principal, `ESC` fecha o jogo.
